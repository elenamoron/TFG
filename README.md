## Proceso de instalación
- Descarga desde el repositorio de GIT
- Instalación de los paquetes javascript necesarios utilizando bower.

```
npm install -g bower
npm install -g polymer-cli  
bower install (instalara todo lo de bower JSON)
polymer serve  
```
